# Bookmark Manager

Веб-приложение для сохранения, организации и быстрого доступа к веб-ссылкам
с использованием категорий и тегов.

Проект выполнен в рамках курса Программирование на языке Python.

---

## Функциональность

### Ссылки
- добавление ссылки (url, название  обязательные, описание  опционально)
- автоматическая дата добавления
- просмотр, редактирование и удаление ссылок
- открытие ссылки в новой вкладке браузера

### Категории
- создание, переименование и удаление категорий
- у каждой ссылки может быть ровно одна категория
- просмотр всех ссылок внутри выбранной категории

### Теги
- создание тегов на лету при добавлении и редактировании ссылки
- у одной ссылки может быть несколько тегов
- просмотр ссылок по тегу
- управление тегами: список, удаление, объединение

### Поиск и фильтрация
- поиск по названию и описанию ссылки
- фильтрация по категории
- фильтрация по одному или нескольким тегам
- комбинированная фильтрация (категория + теги)

### Экспорт
- экспорт данных в формате JSON
- экспорт данных в формате CSV

## Запуск проекта

### Требования
- установлен Python 3.x
- Windows

### Запуск
1. открыть папку проекта
2. запустить файл `start.cmd`
3. выбрать пункт запустить проект
4. открыть в браузере: http://127.0.0.1:5000/
